Digit Optical Character Recognition with Machine Learning

Python

Digit Optical Character Recognition is performed to classify hand-written digits from MNIST data set. I implement k-nearest neighbors - a machine learning algorithm. I first find the distances between the featured vectors in training set and test set. I then sort the calculated distances into ascending order and pick out the top "k" amount of predicted labels. Finally, the number has the highest frequency will be our final predicted label for the test set.
